# Webhooks

Use webhooks to receive real-time notifications when events happen in your TechWriter projects.

## Configuration

1. Go to **Settings > Webhooks** in your dashboard.
2. Enter your payload URL.
3. Select events (e.g., `document.published`).

## Signature Verification

Each request includes a `X-TechWriter-Signature`. Always verify this to ensure the request came from us.
