# Using TechWriter APIs

## Course Overview

Welcome to the comprehensive guide for **TechWriter APIs**. This documentation provides all the necessary information to integrate our documentation and asset management services into your application.

## Introduction to TechWriter APIs

TechWriter APIs are built on REST principles, allowing you to manage documents, users, and workspaces programmatically. Our APIs are designed to be:

* **Scalable**: Handle everything from small projects to enterprise-level document flows.
* **Secure**: Industry-standard encryption and authorization protocols.
* **Developer-friendly**: Detailed error messages and clear resource structures.

In this section, you will learn the core concepts of our architecture and how to make your first successful API request.
