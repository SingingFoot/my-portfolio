### What it is

ChatGPT is a large language model that helps generate, rewrite, and refine text. For technical writers, it acts as a powerful assistant for drafting documentation, improving clarity, and restructuring content.

### How to use it

#### 1. Drafting documentation

You can use ChatGPT to quickly generate initial drafts:

> Write a user guide for installing a CLI tool on macOS.

This gives you a structured starting point that you can refine.

---

#### 2. Simplifying complex text

Paste technical content and ask:

> Rewrite this in simpler language for end users.

This is especially useful for user-facing documentation.

---

#### 3. Creating variations

Need multiple versions?

> Rewrite this for developers / non-technical users.

---

### Best practice

Always review and validate output. ChatGPT accelerates writing, but accuracy still depends on your expertise.
